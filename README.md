# Army Rank Insignia - Merge Puzzle

自衛隊階級モチーフのマージパズルゲームです。スイカゲーム風にピースを落下させ、同じ階級同士を合体させて上位階級へ進化させます。

## 開発言語・技術
- TypeScript
- Phaser 3（Matter.js 物理エンジン）
- Vite

## 起動方法
1. 依存関係をインストール
```bash
npm install
```
2. 開発サーバー起動
```bash
npm run dev
```
3. ブラウザで `http://localhost:5173` を開く

## ビルド
```bash
npm run build
```

## プレビュー
```bash
npm run preview
```

## 操作方法
- クリック/タップ: ピースを落下
- 画面内の左右移動: 落下位置を調整
